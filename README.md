# Chart Visualization Skills

A comprehensive chart generation skill powered by AntV that provides 26 chart types for intelligent data visualization.

## Overview

Chart Visualization Skills intelligently selects the most appropriate chart type from 26 available options, extracts parameters based on detailed specifications, and generates high-quality chart images. It covers time series, comparisons, part-to-whole, relationships, geographic, hierarchical, statistical, and specialized visualizations.

## Usage

Add this skill to your project:

```bash
npx skills add antvis/chart-visualization-skills
```

## Available Chart Types

- **Time Series**: Line, Area, Dual Axes
- **Comparisons**: Bar, Column, Histogram
- **Part-to-Whole**: Pie, Treemap
- **Relationships**: Scatter, Sankey, Venn, Network Graph
- **Geographic**: District Map, Pin Map, Path Map
- **Hierarchies**: Organization Chart, Mind Map
- **Statistical**: Box Plot, Violin
- **Specialized**: Radar, Funnel, Liquid, Word Cloud, Fishbone Diagram, Flow Diagram, Spreadsheet

For detailed specifications of each chart type, see the [`references/`](references/) directory.
